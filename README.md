# subms_ja

https://www.reddit.com/r/newsokur/wiki/subreddits_jp からサブレを抽出する.

## Installation

1. Python3をインストールする.
2. pip install https://github.com/demmc/subms_ja/archive/master.tar.gz

## Example

```sh
$ subms_ja
100kin
AdminTrans_ja
Android_JP
AnimeFigures_ja
Aquariums_ja
AskReddit_ja
BABYMETAL_japan
BakaNewsJP
BayStars
CG_ja
...
```
